Detailed Description

File containing the implementation of the RGB LED control library.

Author
Massimo Dalla Bona
Date
04.07.2023
Version
1.0

This library provides an interface to control multiple RGB LEDs independently on an ESP32 microcontroller. It utilizes the microcontroller's GPIO and LEDC functionalities to provide precise control over the color and brightness of each LED. The library supports the configuration of each LED's color, as well as enabling various lighting effects including blinking, transition, and breathing effects, with customizable timing parameters. Each RGB LED can be individually controlled, allowing for complex light effects across multiple LEDs.

Note
This implementation is compatible with the ESP-IDF development framework, version 5.0.2.

esp_err_t rgb_led_init ( rgb_led_t *  rgb_led_cfg)

Initialize the LEDC to control an RGB LED.

@description This function initializes the ESP32's LEDC (LED Controller) hardware peripheral to control an RGB LED. The RGB LED is represented by an rgb_led_t struct which contains the GPIO numbers and LEDC channels for the red, green, and blue LEDs. The function first checks if the provided GPIO numbers are valid for output, and if the LEDC channels are not already in use. It then configures the GPIOs for output and sets up the LEDC peripheral to control the LEDs. The function returns ESP_OK if the initialization was successful. If there was an error, it returns an appropriate ESP_ERR code.

Parameters
rgb_led_cfgPointer to an rgb_led_t struct containing the configuration data for the RGB LED.
Returns
esp_err_t ESP_OK if successful, or an error code if not.

Set the color of an RGB LED.

@description This function sets the color of an RGB LED by setting the duty cycle of the LEDC channels controlling the red, green, and blue LEDs. The color is specified as a 24-bit RGB color value. The red, green, and blue components are extracted from the color value, and the LEDC duty cycle is set proportional to the component value. Since the LEDC timer resolution is 8 bits, the color component values are used directly as the duty cycle. If an error occurs while setting the duty cycle, an appropriate ESP_ERR code is returned.

Parameters
rgb_ledA pointer to the rgb_led_t structure that contains the configuration of the RGB LED.
colorThe 24-bit RGB color value. The red component is in the most significant 8 bits, followed by green, and then blue.
Returns
esp_err_t ESP_OK if successful, or an error code if not.

Start a blink effect for the specified LED.

This function starts a blink effect for the specified LED with the given parameters. The LED will alternate between the specified color and off state for the specified number of pulses. The time_on parameter specifies the duration in milliseconds for which the LED will be on, while the time_off parameter specifies the duration in milliseconds for which the LED will be off. The time_out parameter specifies an optional timeout period in milliseconds after which the blink effect will stop automatically.

Parameters
ledPointer to the LED structure.
colorThe color to blink (RGB value).
pulse_countThe number of pulses for the blink effect.
time_onThe duration in milliseconds for which the LED will be on during each pulse.
time_offThe duration in milliseconds for which the LED will be off during each pulse.
time_outThe optional timeout period in milliseconds after which the blink effect will stop automatically. Set to 0 to disable timeout.
Returns
  • ESP_OK if the blink effect was successfully started.
  • ESP_ERR_INVALID_ARG if the input argument is invalid.
  • ESP_FAIL if the LED is not found in the LED effects array or the maximum number of LED effects is reached.
  • Other error codes if there was an error starting the timer.

Set the color of an RGB LED with a smooth transition effect.

This function sets the color of an RGB LED and smoothly transitions it to the new color over the specified transition duration. The function takes the RGB LED configuration, the current color, the new color, and the duration of the transition in milliseconds. It sets the LED to the new color and creates a timer to gradually transition it back to the previous color after the specified duration. The previous color is stored and used in the transition callback function.

Parameters
rgb_ledPointer to the rgb_led_t structure that contains the configuration of the RGB LED.
current_colorThe current color of the LED in RGB format (e.g., 0xRRGGBB).
new_colorThe new color to transition the LED to in RGB format.
effect_durationThe duration of the color transition in milliseconds.
Returns
  • ESP_OK if successful
  • ESP_FAIL if a transition is already in progress
  • Other error codes if an error occurred

Set the color of an RGB LED with a smooth transition effect.

This function transitions the RGB LED from the current color to the new color over a specified duration.

Parameters

  • ESP_ERR_INVALID_ARG if the LED pointer is invalid.
